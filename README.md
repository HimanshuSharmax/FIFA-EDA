# ⚽ FIFA 20 Player Data Exploratory Data Analysis (EDA)

## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on the **FIFA 20 Player Dataset** using **Python**. The objective is to analyze player statistics, salaries, nationalities, and performance metrics while extracting meaningful insights through data visualization and statistical analysis.

## 🚀 Objectives

* Analyze player distribution across different nationalities.
* Identify the highest-paid football players.
* Compare player attributes such as **height, weight, wages, and shooting ratings**.
* Perform country-specific analysis (e.g., Germany).
* Visualize trends using informative charts.

## 📊 Dataset

* **Dataset:** FIFA 20 Player Dataset (`players_20.csv`)
* **Records:** 18,000+ players
* **Attributes:** 100+ player features

## 🛠️ Tech Stack

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## 🔍 Analysis Performed

* Data loading and exploration
* Nationality distribution analysis
* Player salary analysis
* Player ranking based on wages
* Country-wise player analysis
* Height and weight comparison
* Shooting rating analysis
* Data visualization using bar charts

## 📈 Key Insights

* Identified the **top five nationalities** with the highest number of players.
* Ranked players based on **wages** to determine the highest-paid footballers.
* Compared players using **height, weight, and shooting ratings**.
* Performed detailed analysis of **German players** across multiple performance metrics.
* Generated visualizations to simplify trend identification and support data-driven insights.
